How to Use

1. Open the Review Agent and enter your story or Jira issue key.

2. Add your Langflow API key in the field above.

3. Click Review Story and wait for the agent to analyze the story.

4. Copy the AI review result or clear the chat to start again.

This page calls your local Langflow flow directly and displays results inside the chat window.

Sample Fund Transfer User Story
As a customer, I want to transfer funds between my accounts, So that I can manage my finances without visiting a branch. Acceptance Criteria: • User can select source account • User can select destination account • User can enter transfer amount • Amount must be greater than zero • Confirmation message should be displayed • Transaction should be saved successfully
